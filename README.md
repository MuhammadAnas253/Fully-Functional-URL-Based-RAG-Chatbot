Fully Functional URL-Based RAG Chatbot
A Retrieval-Augmented Generation (RAG) chatbot that allows users to input a website URL and ask questions about the content of that webpage. The system extracts the webpage data, converts it into embeddings, and enables intelligent question answering using a language model.
This project demonstrates a practical implementation of RAG architecture using modern LLM tooling.

**Project Overview**
This application allows users to:
Provide a URL of any webpage
Extract and process webpage content
Convert content into vector embeddings
Store embeddings in a vector database
Ask natural language questions about the webpage
Receive context-aware answers generated by an LLM

The system follows the standard RAG pipeline used in production AI systems.

**RAG Workflow**
The system works in the following steps:
User provides a website URL
The application loads webpage content using a URL Loader
Text is cleaned and split into chunks
Chunks are converted into vector embeddings
Embeddings are stored in a vector database

**When a question is asked:**
Relevant chunks are retrieved
Context is passed to the LLM
The LLM generates an accurate answer
